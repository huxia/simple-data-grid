Paging example for simple-data-grid

This example shows how you can use pages.

Name Latin name

JSON response

The JSON response from the server should contain the following properties:

total_pages: number of total pages in the data
rows: rows in this page; this is an array of objects 

{
  "total_pages": 10,
  "rows": [
    {
      "latin_name": "Persea americana",
      "name": "Avocado"
    },
    {
      "latin_name": "Capsicum annuum",
      "name": "Bell pepper"
    }
  ]
}
